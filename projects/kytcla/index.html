Kytcla - Educación Contínua

Descripción

Aplicación móvil educativa y comercial desarrollada para Kytcla, diseñada para promover cursos de formación continua y productos especializados. La plataforma ofrece una experiencia intuitiva para explorar, guardar y solicitar información sobre ofertas académicas y comerciales, priorizando la interacción directa con el equipo de ventas a través de WhatsApp.

Al no incluir un sistema de compras en línea, la aplicación se enfoca en la generación de leads y el acompañamiento personalizado, permitiendo a los usuarios inscribirse en cursos, guardar sus intereses y contactar de forma inmediata al responsable correspondiente. Desarrollada en Android con Java y backend en PHP/MySQL, combina funcionalidad técnica con una propuesta de valor centrada en la cercanía y el servicio humano.

Componentes

  • Gestión de Cursos

    Permite las operaciones básicas de crear, modificar, eliminar y listar cursos. Adicionalmente cuenta con opciones para completar un formulario para inscribirse y solicitar mayor información por medio de WhatsApp.


    Al momento de crear un curso, solo es posible agregar una imágen que corresponde al logo del curso.

  • Gestión de Productos

    Permite las operaciones básicas de crear, modificar, eliminar y listar productos. Adicionalmente cuenta con opciones para solicitar mayor información por medio de WhatsApp.


    La aplicacion no gestiona la compra en línea, por lo que si a un usuario le interesa un producto, debe contactarse por medio de WhatsApp con el/la responsable de ventas. Existe un botón en la descripción de cada producto para que pueda ponerse en contacto directamente con el/la responsable de ventas.


    Al momento de crear un producto, solo es posible agregar una imágen por producto.

  • Gestión de Elementos Guardados

    Cuando el usuario este viendo los detalles de un curso o un producto, tiene la posibilidad de guardarlo para separarlo como sus elementos preferentes.

  • Gestión de Perfil

    El usuario tiene la posibilidad de modificar los datos de su perfil público.

  • Gestión de Cuenta

    El usuario tiene la posibilidad de modificar su contraseña.

Tecnologías Utilizadas

Android Java PHP MySQL
